Как устроен сайт — аналогия с домом

Представьте, что создание сайта — это строительство дома. Каждый слой добавляет новый уровень.

Только HTML

Это скелет дома — фундамент, стены, перекрытия, перегородки. Дом уже существует, в нём можно жить, но выглядит он как стройка.

Фундамент

HTML-теги — это фундамент. <h1>, <p>, <img> — каждый тег несёт свою нагрузку

Стены и перегородки

Структура страницы: шапка, контент, боковая панель, подвал — всё разбито на логические блоки

Двери и окна

Ссылки и кнопки — это двери и окна. Они ведут куда-то или открывают что-то

HTML + CSS — дом с отделкой

Добавляем CSS — и дом из стройки превращается в уютное жильё.

HTML + CSS

CSS — это отделка: покраска стен, обои, напольное покрытие, светильники. Дом становится красивым и комфортным.

Цвета и материалы

Фон, цвета текста, границы — CSS задаёт палитру всего дома, как краска и обои

Шрифты и размеры

Размер заголовков, межстрочные интервалы — как выбор светильников и декора

Расположение элементов

Отступы, сетка, flexbox — как расстановка мебели в уже готовых комнатах

HTML + CSS + JS — умный дом

Добавляем JavaScript — и дом становится умным, живым, реагирует на действия.

HTML + CSS + JS

JS — это автоматизация и интерактив: кнопки света, автоматические шторы, датчики движения. Дом начинает «думать».

Кнопки и события

Нажал кнопку — включился свет. Клик по кнопке — открылось меню, появился попап

Анимации и переходы

Плавное появление блоков при скролле, слайдеры, выпадающие списки — дом «оживает»

Данные и логика

Формы, валидация, загрузка данных — дом принимает решения на основе того, что происходит

Что такое ИИ и как строить промпт

ИИ — это не волшебство. Это инструмент, который понимает инструкции. Чем точнее инструкция — тем лучше результат. Эта инструкция называется промпт.

1

Роль

Кем должен быть ИИ? «Действуй как веб-разработчик»

2

Задача

Что нужно сделать? «Напиши код для лендинга»

3

Контекст

Какие данные? Название, тематика, аудитория, структура

4

Формат

В каком виде ответ? Один HTML-файл, адаптивный, с анимациями

5

Ограничения

Цвета, шрифты, стиль — всё, что важно для результата
